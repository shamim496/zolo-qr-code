== Installation ==

= Plugin Installation Method: =

1. Go to the WordPress dashboard
2. Got to the **Plugins** > **Add New**
3. Type **"QR Code"** in the search box
4. Click on **"Install"** button
5. After install the plugin **Activate** the plugin

= Installation via Zip file: =

1. Download QR Code block **plugin zip** file from WordPress
2. Go to site **Dashboard** > **Plugins** > **Add New** > **Upload Plugins**
3. Select the QR Code zip file and proceed to install
4. Activate QR Code.

== Frequently Asked Questions ==

= How to Use =

1. **Install and Activate the Plugin**  
   - Upload the plugin files to `/wp-content/plugins/qr-code-block` or install it via the WordPress Plugins screen.  
   - Activate the plugin through the 'Plugins' screen in WordPress.

2. **Add the QR Code Block**  
   - Open the page or post where you want to display the QR code.  
   - Click the "+" button to add a block in the Gutenberg editor.  
   - Search for "QR Code" and select the "QR Code Block" from the list.

3. **Customize the QR Code**  
   - Enter the URL, text, or other content you want the QR code to represent in the block settings.  
   - Adjust the size, color, and background as needed.  
   - Preview the QR code to ensure it appears as desired.

4. **Publish or Update the Page**  
   - Once youâ€™re satisfied with the QR codeâ€™s appearance, click "Publish" or "Update" to make it live on your website.

Your QR code is now live and ready to use! Users can scan it with their devices to access the linked content or information.
